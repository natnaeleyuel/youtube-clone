# youtube-clone

A static clone of the YouTube homepage, built using only HTML and CSS. This project demonstrates how to replicate a popular web interface with semantic markup and modern styling techniques.

## Contents

- **index.html:** Main webpage replicating YouTube’s layout.
- **src/**: Supporting styles and assets for the site.
- **README.md:** Project overview and usage instructions.

## Features

- Pixel-perfect layout inspired by YouTube.
- Responsive design for various screen sizes.
- No JavaScript—purely HTML and CSS.
- Live demo: [https://natnaeleyuel.github.io/youtube-clone/](https://natnaeleyuel.github.io/youtube-clone/)

## Usage

1. Clone or download this repository:
   ```sh
   git clone https://github.com/natnaeleyuel/youtube_clone.git
   ```
2. Open `index.html` in any modern web browser to preview the site.
